# CoverageJSON / BioClima Coverage

Family block for the BioClima CoverageJSON profile. It groups the reusable CoverageJSON constraints used by EBV datasets, derived indicators and zonification outputs.

## Where this appears

- Interactive map viewer: <https://maytetoscano.github.io/bblocks-bioclima/viewer/>
- Classified register by family: <https://maytetoscano.github.io/bblocks-bioclima/viewer/registry.html#fam-coveragejson>
- OGC Building Blocks viewer: <https://maytetoscano.github.io/bblocks-bioclima/bblocks/bblock/ogc.bioclima.coveragejson>
