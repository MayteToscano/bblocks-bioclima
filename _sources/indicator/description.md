# Spatial Indicators

Family block for spatial indicators. It groups the generic indicator recipe and concrete indicators derived from EBV datasets, including the Phenological Change Indicator.

- Interactive map viewer: <https://maytetoscano.github.io/bblocks-bioclima/viewer/>
- Classified register by family: <https://maytetoscano.github.io/bblocks-bioclima/viewer/registry.html#fam-indicator>
- OGC Building Blocks viewer: <https://maytetoscano.github.io/bblocks-bioclima/bblocks/bblock/ogc.bioclima.indicator>
