# Bioclimatic Zonification

Family block for bioclimatic zonification. It groups polygon-based classification outputs derived from indicators and intended for decision-support use.

## Where this appears

- Interactive map viewer: <https://maytetoscano.github.io/bblocks-bioclima/viewer/>
- Classified register by family: <https://maytetoscano.github.io/bblocks-bioclima/viewer/registry.html#fam-zonification>
- OGC Building Blocks viewer: <https://maytetoscano.github.io/bblocks-bioclima/bblocks/bblock/ogc.bioclima.zonification>
