## Description

# BioClima ontology

This building block publishes the semantic model used by the BioClima register.

It defines the concepts referenced by EBVs, observed properties, spatial indicators, policy-alignment records, provenance records and bioclimatic zonification categories. The concepts are expressed in RDF/SKOS and supported by SHACL shapes.
